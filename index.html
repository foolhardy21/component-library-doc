akira ui

here are the simple steps to import the library in your project.

import the stylesheet as mentioned below in the head of the html document. 

            
            <link rel='stylesheet' href='https://deploy-preview-10--festive-mestorf-acecb0.netlify.app/stylesheet/main.css' />

along with the stylesheet you would need to import material icons.

            
            <link href="https://fonts.googleapis.com/icon?family=Material+Icons" rel="stylesheet" />

that's it. refer the documentation to know more.

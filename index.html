badges

small notification badges

                
                <div class='badge-size-sm badge-green border-radius-circle margin-btm-sm'></div>

badge-size-sm is for small notification badges. It gives width and height of 0.7rem to the div. For larger sizes, refer the examples shown below this.

badge-blue is for the fill color of the badge. Replace -blue with other options i.e -red, -green or -yellow.

border-radius-circle gives the border radius of 50%.

medium notification badges

                
                <div class='badge-size-md badge-green border-radius-circle margin-btm-sm'></div>

Everything is same as above with the only difference being the class badge-size-md. It gives width and height of 1rem to the div.

Buttons

text buttons

                
                <button class='btn-text text-blue container-transparent no-border font-size-sm padding-md text-uppercase text-bold letter-space-sm border-radius-sm'>
                    text
                </button>

btn-text sets border-style to none and sets the background to transparent. It also sets the font-color.

border radius-sm sets the border-radius to 5px.

On hover the background color is updated using :hover pseudo class.

contained buttons

                
                    <button class='btn-filled text-white no-border font-size-sm padding-md text-uppercase text-bold letter-space-sm border-radius-sm'>
                        filled
                    </button>

btn-filled sets the background and font color and sets the border-style to none.

On hover, the behaviour is same as text button, only the background color set is different.

outlined buttons

                
                <button class='btn-outlined text-blue border-blue border-thin container-transparent text-uppercase text-bold font-size-sm padding-md letter-space-sm border-radius-sm'>
                    outlined
                </button>

btn-outlined sets a solid border with thin border-width and sets a transparent background.

buttons with icons

icons with contained buttons

                
                <button class='btn-filled flex-center text-uppercase text-bold padding-md font-size-sm letter-space-sm border-radius-sm margin-btm-sm'>
                    <span class='material-icons'>
                        delete
                    </span>
                    delete
                </button>

flex-center sets the contents of button at the center of major and cross axis.

icon button

                
                <button class='btn-text text-blue container-transparent no-border padding-lg flex-center border-radius-circle'>
                    <span class='material-icons'>alarm</span>
                </button>

border-radius-circle makes the border-radius of 50%.

padding-lg sets the padding of 0.6rem.

:hover is used to update the background color.

functioning float action button is in the bottom right corner.
